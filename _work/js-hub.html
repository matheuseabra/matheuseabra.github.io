State of the art JavaScript tooling 🔨


JS Hub is a open-soruce project where there's a simple grid displaying the top repositories for major technologies within the JS ecosystem. It includes major frameworks, libraries and other not so know but still interesting repositories. The project is a single-page application built in React as the UI, using the Github API and Browser's localStorage for dark mode.

My role in this project


I was responsible for the full cycle of Design and Development, as well as Deployment and maintenance of the codebase in production.
